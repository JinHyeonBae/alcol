# alcol(알코올 레시피 홈페이지)

## 목차
- 구조
- 기술 스택
- 결과

## 구조
술은 맛이 없으므로 맛있게 만들자라는 생각으로 만든 홈페이지이다. 술에 대한 정보를 알려주는 사전 페이지와, 알코올 레시피에 대한 글을 작성할 수 있고, 추천을 받을 수 있으며 홈에서 현재 좋아요를 가장 많이 받은 순위도 볼 수 있다. 


## 기술 스택
- JSP
